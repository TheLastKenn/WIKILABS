Convierte fracciones complejas en expresiones simples

Utiliza estos pasos para limpiar fracciones dobles o anidadas y asegurarte de mantener el dominio original.

1. Identifica subfracciones

Observa si el numerador o denominador contiene fracciones. Reescribe cada subfracción como división explícita.

2. Encuentra el MCM de subfracciones

Multiplica numerador y denominador por el mínimo común múltiplo de los denominadores internos para eliminarlos.

3. Simplifica la expresión resultante

Una vez sin fracciones internas, se convierte en una fracción algebraica usual. Aplica simplificación adicional si procede.

Ejemplos comentados

Observa cómo se simplifica paso a paso

Ejemplo básico

  • (x/(x + 1))/(x - 2).
  • Multiplica numerador y denominador por x + 1.
  • Resultado: x/[(x + 1)(x - 2)] con x ≠ -1, 2.

Dos niveles

  • (1 + 2/x)/(3 - 4/x).
  • Multiplica por x y obtén (x + 2)/(3x - 4).
  • Restricciones: x ≠ 0 y x ≠ 4/3.

Con variables múltiples

  • ((a/b) + (b/a))/((1/a) - (1/b)).
  • Multiplica por ab para eliminar fracciones internas.
  • Resultado final: (a^2 + b^2)/(b - a) con a, b ≠ 0.
