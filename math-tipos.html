Fracciones propias

  • El grado del numerador es menor que el del denominador: (x + 1)/(x^2 + 4).
  • Su valor absoluto es menor que 1 cerca del infinito.
  • Son ideales para modelar tasas decrecientes o amortiguadas.

Fracciones impropias

  • El grado del numerador es igual o mayor al del denominador.
  • Se pueden reescribir como polinomio + fracción propia mediante división sintética.
  • Practica con (x^2 + 2x + 1)/(x + 1) y realiza la división para descubrir su parte polinómica.

Aparentes y complejas

  • Aparente: el denominador divide exactamente al numerador y la expresión se simplifica a un polinomio.
  • Compleja: contiene otras fracciones en numerador, denominador o ambos.
  • Requieren especial cuidado con las restricciones al simplificar.

Comparador de comportamiento

Utiliza estas referencias para anticipar cómo se comporta cada categoría a medida que x crece o tiende a valores críticos.

Propia

Tiende a 0 cuando |x| → ∞. Ideal para describir sistemas que se estabilizan.

Impropia

Puede divergir o estabilizarse dependiendo de factores ocultos; se recomienda dividir los polinomios para leerla con claridad.

Aparente

Se simplifica completamente, pero conserva las restricciones originales. Úsala para detectar oportunidades de reducción.

Compleja

Transforma expresiones anidadas en fracciones comunes mediante el truco de multiplicar por el inverso adecuado.
