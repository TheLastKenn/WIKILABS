Paso a paso

Procedimiento para hallar el dominio

Sigue esta ruta para identificar rápidamente qué valores debe evitar tu variable. Úsala antes de simplificar o resolver ecuaciones.

1. Igualar a cero el denominador

Resuelve Q(x) = 0. Cada solución es un valor prohibido para la expresión original.

2. Validar restricciones adicionales

Si existen raíces o logaritmos en el denominador, asegúrate de considerar condiciones como positividad o radicandos no negativos.

3. Registrar el dominio

Escribe el conjunto de valores válidos usando notación de intervalo o conjunto, excluyendo los puntos prohibidos encontrados.

4. Marcar comportamiento gráfico

Identifica si los valores prohibidos generan discontinuidades removibles o asíntotas verticales al analizar el gráfico.

Casos especiales

Cuando el denominador tiene factores ocultos

Explora situaciones donde el denominador incluye potencias, raíces o productos de binomios. Cada caso requiere una reflexión diferente para delimitar el dominio.

Factores lineales

  • Ejemplo: 1/(x - 4). Determina qué valor vuelve cero al denominador y anótalo como restricción.
  • Se producen asíntotas verticales en los valores prohibidos.

Factores cuadráticos

  • Ensaya con 1/(x^2 - 9): factoriza para resolver las raíces reales del denominador.
  • Siempre factoriza para encontrar raíces reales.

Radicales en el denominador

  • 1/√(x - 2) exige que el radicando sea positivo para evitar valores no reales.
  • Extrae la condición sin escribir la solución final y recuerda excluir el valor que hace cero al denominador.
