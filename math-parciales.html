Metodología

Paso a paso para descomponer

Utiliza este marco para convertir fracciones impropias en sumas manejables.

1. Verificar grados

Si el grado del numerador es mayor o igual al del denominador, realiza división de polinomios para obtener una fracción propia.

2. Factorizar el denominador

Expresa Q(x) como producto de factores lineales y/o irreducibles.

3. Plantear suma de fracciones simples

Asigna coeficientes a determinars: A/(x - r), (Bx + C)/(x^2 + px + q), etc.

4. Resolver coeficientes

Iguala numeradores o evalúa valores estratégicos para hallar los coeficientes.

Ejemplos guiados

Casos representativos

Factores lineales distintos

  • 3x/(x^2 - 1) → A/(x - 1) + B/(x + 1).
  • Resultado: (3/2)/(x - 1) + (3/2)/(x + 1).

Factores cuadráticos

  • (2x^2 + 5x + 7)/[(x + 2)(x^2 + 1)].
  • Plantea: A/(x + 2) + (Bx + C)/(x^2 + 1).

Factores repetidos

  • 5/(x - 1)^2 → A/(x - 1) + B/(x - 1)^2.
  • Recuerda incluir términos para cada potencia del factor repetido.
